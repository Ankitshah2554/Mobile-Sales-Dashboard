# Mobile Sales Dashboard - Power BI Project
## Project Overview
This project is a Mobile Sales Analysis Dashboard created using Power BI.
The dashboard helps analyze:
- Total Sales
- Total Quantity
- Transactions
- Average Sales
- Customer Ratings
- Payment Methods
- City-wise Sales
- Mobile Brand Performance
- Daily Sales Trends
- ---
## Tools Used
- Power BI
- Excel
- Data Visualization
- DAX
- Power Query
- ---
## Dashboard Features
- Interactive Filters (Month, Brand, Customer)
- KPI Cards
- Sales Trend Analysis
- Payment Method Analysis
- Customer Rating Insights
- City-wise Sales Map
- Brand Comparison
---
## Dataset
The dataset contains:
- Mobile Brands
- Sales Amount
- Quantity Sold
- Customer Ratings
- Transactions
- Payment Methods
- Cities

## Key Insights
- Apple and Samsung generated highest sales.
- UPI and Debit Card were major payment methods.
- Sales were highest on weekends.
- Mumbai and Delhi showed strong sales performance.
